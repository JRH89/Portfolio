    COVIDcounter is a webpage that displays the number of confirmed cases, deaths, new cases, and mortality rate for almost every country using data from the public Covid19API. Because of the strict rate limiting on the data calls, this page is prone to displaying an error. If enough people download it once it is on the Google Play store, I will subscribe to a paid tier API service to provide a better user experience. I have also converted it into an .apk for Android.
