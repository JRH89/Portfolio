Recent Projects:

  • Weather App

    This weather app utilizes the OpenWeather API to retrieve current weather data for any city and display it in a user-friendly format. I have also successfully built an APK that functions well on Android OS.


  • COVIDcounter

    COVIDcounter is an app that visualizes the number of confirmed cases, deaths, new cases, and mortality rate for almost every country using data from a public API. Because of the strict rate limiting on the data calls, this app is prone to displaying an eror. If enough people download it once it is on the Google Play store, I will subscribe to a paid tier API service to provide a better user experience.


  • Robotic Seahorse JS Game

    Keyboard required. A 2D side-scroller game made with JavaScript where the player controls a robotic seahorse, shooting and defeating enemies with lasers to earn enough points before the timer runs out.


  • Ghost Dog JS Game

    Keyboard required to play. A simple JavaScript game where you play as a ghost dog trying to save the forest from evil creatures. Turn into a ball of fire and roll over or jump onto enemies to destroy them!


  • Interactive Particle Text

    This project is an interactive particle text effect made with HTML, CSS, and JavaScript. Type something into the text bar and move your mouse cursor across the rendered text to send particles scattering before they return to their rightful places. This can be customized in thousands of ways to make it unique for many different applications.


  • Drum Kit

    Keyboard required. Not mobile friendly yet. A simple drum kit made with JavaScript, HTML, and CSS, where you can play sounds by clicking or pressing the keyboard keys.
